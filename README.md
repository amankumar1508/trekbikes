# Trek Bikes Clone 🚲

A high-fidelity, responsive static website clone of the **Trek Bikes** official site. This project showcases modern web design techniques using Vanilla HTML5 and CSS3, featuring a premium user interface with video backgrounds, interactive components, and a clean, developer-centric directory structure.

## ✨ Features
- **Responsive Navigation**: Dual-logo header (Trek & Bontrager) with a mobile-friendly hamburger menu.
- **Dynamic Header**: Immersive video background for the hero section, featuring the latest Trek arrivals.
- **Multi-page Experience**:
  - **Home**: Main showcase of bikes, gear, and brand stories.
  - **All-New Arrivals**: Dedicated page for the 2026 bike lineup and equipment.
  - **Become a Dealer**: Interactive sign-up landing page for prospective retail partners.
- **Tribute Design**: Closely replicates the premium aesthetic of the official Trek Bikes global site.
- **Newsletter Integration**: Fully styled newsletter signup section in the footer.
- **Social Connect**: Integrated social media links and regional selectors.

## 🛠️ Technologies Used
- **HTML5**: Semantic structure for SEO and accessibility.
- **CSS3**: Custom styling, layout (Flexbox/Grid), and animations without external frameworks.
- **Cloudinary**: Optimized video hosting for high-performance background playback.
- **Base64 Assets**: Efficient SVG icon embedding to reduce HTTP requests.

## 🛠️ Setup and Usage

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge).
- A local server (recommended for video playback), such as **VS Code Live Server**.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-/trekbikes.git
   ```
2. Navigate to the project directory:
   ```bash
   cd trekbikes
   ```

### Running Locally
- Simply open `index.html` in your browser.
- For the best experience (including video gradients and smooth transitions), use a local development server like the **Live Server** extension in VS Code.
